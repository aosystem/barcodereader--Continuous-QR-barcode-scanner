## 🌟 Why This App?

Unlike standard camera apps, this tool is optimized for **speed** and **batch processing**.

- **Continuous Scanning:** Scan multiple items one after another without interruption.
- **CSV Data Export:** Instantly share your scan history via CSV for Excel or Google Sheets.
- **Global Support:** Fully localized in **50+ languages**, supporting users worldwide.
- **Privacy Focused:** Open-source core logic. No hidden data collection or tracking.

## 💡 Best For...
- **Inventory & Stocktaking:** Streamline warehouse management.
- **Event Management:** Quick attendee check-ins at entrances.
- **Asset Tracking:** Manage equipment and tools for your business.

## 🛠 Technical Highlights
- Built with **Flutter** for high performance across platforms.
- Core logic for high-speed barcode recognition is open-sourced here.
- *Note: This repository contains the core logic only. Assets (images, audio, and localization files) are excluded to protect intellectual property.*
